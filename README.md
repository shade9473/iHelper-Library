# iHelper Resource Library V2

## 📘 Overview
A comprehensive professional resource library with dynamic content mapping, advanced search capabilities, and enhanced accessibility features.

## 🚀 Quick Start

### Prerequisites
- Node.js 20.x
- npm 9.x or higher
- Git

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/ihelper-resource-library.git
cd ihelper-resource-library

# Install dependencies
npm install
```

### Development Workflow
```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm test

# Lint code
npm run lint
```

## 🔍 Key Features
- 📚 Dynamic Resource Categorization

## 🛠 Technology Stack
- **Frontend**: Vue 3
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: Pinia
- **Routing**: Vue Router
- **Performance**: Rollup Visualizer
- **Deployment**: Cloudflare Pages
- **CI/CD**: GitHub Actions

## 📦 Deployment Configuration

### Environment Variables
Create a `.env` file with the following:
```
VITE_ENVIRONMENT=production
VITE_PERFORMANCE_MODE=optimized
```

### Cloudflare Pages Deployment
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Node Version**: `20.x`

### GitHub Actions Workflow
Automatic deployment is configured for:
- Push to `main` branch
- Pull requests to `main`
- Manual workflow dispatch
